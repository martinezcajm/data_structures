◆ concat

s16(* adt_list_ops_s::concat) (List *list, List *src)

Concatenates two list storing the result at origin.

Concatenates the source to the list, in case one of the list has infinite capacity the result list will have infinite capacity, otherwise it will be the result of both capacities added. Notice that the src list won't be modified during the execution. In case the list is null kErrorCode_Null_List will be returned and in case the the src is null kErrorCode_Null_Pointer_Parameter_Received. If there's a problem during the allocation of the new list kErrorCode_Error_Trying_To_Allocate_Memory will be returned.

Returns
s16 status of the operation once finished
Parameters
*listpointer to the list that will store the result of the concatenation
*srclist we want to concatenate to our origin
